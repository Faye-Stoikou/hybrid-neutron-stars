# Hybrid Neutron Stars and Hadron–Quark Phase Transition

This repository contains numerical work related to my undergraduate thesis on hybrid neutron stars and the hadron-to-quark matter transition.

## Part 1 – CSS Model

Hadronic matter described by the APR-1 equation of state and quark matter modeled using the Constant Speed of Sound (CSS) model.  
The phase transition is implemented using a Maxwell construction.

The Tolman–Oppenheimer–Volkoff equations are solved numerically to obtain mass–radius relations and investigate the appearance of twin star solutions.

## Part 2 – Self-consistent Transition

A self-consistent determination of the transition pressure is developed using thermodynamic quantities of APR-1 and the color–flavor–locked (CFL) model.

Hybrid equations of state are constructed and the TOV equations are solved to explore the structure of hybrid neutron stars.
